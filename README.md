## How to setup Skin
1. Open tab from "File > Preferences > Settings"
2. Select "Preview > Mincraft Skin"
3. Select File "skin_slim_alex.png" in project

## How to setup plugin
1. Open tab from "File > pluins"
2. Load Plugin from file in project
