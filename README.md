# CosmeticsCore Editor

A Blockbench plugin for creating and editing CosmeticsCore cosmetics with live preview and advanced tools.

## 📥 Installation

### Option 1: Download from Releases (Recommended)
1. Go to [Releases](../../releases) page
2. Download the latest `cceditor.js` file  
3. Open Blockbench
4. Go to **File** → **Plugins** → **Load Plugin from File**
5. Select the downloaded `cceditor.js` file

### Option 2: Load from Project
1. Clone this repository
2. Open Blockbench
3. Go to **File** → **Plugins** → **Load Plugin from File**
4. Select `cceditor.js` from the cloned folder

## ⚙️ Setup

### Minecraft Skin Setup
1. Open **File** → **Preferences** → **Settings**
2. Navigate to **Preview** → **Minecraft Skin**
3. Select the `skin_slim_alex.png` file from this project

### Plugin Configuration
1. After loading the plugin, you'll see "CosmeticsCore" in the menu bar
2. Follow the setup wizard for your specific cosmetic type
3. Refer to the [CosmeticsCore Wiki](https://cosmeticscore.devs.beer/adding-content/body-cosmetic/static-body-cosmetic) for detailed instructions
